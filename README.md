ABSTRACT
This project focuses on the design and development of an educational website called Digital Skills Hub. The website was created to help students acquire basic digital and computer skills needed in today’s digital world.

The project was developed using HTML, CSS, and JavaScript without the use of any WYSIWYG web authoring tools. The website consists of five interconnected pages and includes appropriate multimedia elements. GitHub Pages was used to host the website online.

This report documents the entire development process, including planning, design, implementation, challenges faced, and solutions applied.

CHAPTER ONE: INTRODUCTION
1.1 Background of the Study

The use of digital technology has become a major part of everyday life, especially in education. Students are required to use computers, mobile devices, and the internet for research, communication, and academic work. Despite this, many students still lack basic digital skills.

The Digital Skills Hub website was developed to provide students with simple and easy-to-understand digital knowledge. The website focuses on teaching essential skills such as computer usage, internet navigation, document creation, and online safety.

1.2 Problem Statement

Many students face challenges when using computers and online tools due to limited digital knowledge. This affects their academic performance and confidence. There is therefore a need for a simple online platform that provides basic digital skills education.

1.3 Aim and Objectives

Aim

The aim of this project is to design and develop an educational website that teaches essential digital skills.

Objectives

To design a multi-page website using HTML and CSS

To provide appropriate educational content

To interlink all web pages using functional links

To include suitable multimedia elements

To host the website using GitHub Pages

1.4 Scope of the Project

The project focuses on frontend website development only. Backend systems such as databases and user authentication were not included. The website is intended for educational purposes.
